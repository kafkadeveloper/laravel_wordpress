# Using Laravel 5 within WordPress (and the other way around)



## Tutorial 1: Installing

1. Boot your vagrant VM by running :

```
vagrant up
```


2. Add the following entry in your /etc/hosts file

```
10.10.10.10 site.vagrant
```
